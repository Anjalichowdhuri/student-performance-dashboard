# Student Performance Dashboard (Excel)

## 📊 Project Overview
This project analyzes student performance and demographic data using Excel. It provides insights into academic trends, attendance, and category-wise performance.

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Graphs
- Data Cleaning & Filtering

## 📈 Key Insights
- Year-wise student performance trends
- Attendance analysis
- Category-wise comparison of scores
- Identification of high and low performers

## 📂 Files Included
- scse_2023_hostel_updated real.xlsx → Dataset and analysis

## 🚀 How to Use
1. Download the Excel file
2. Open in Microsoft Excel
3. Explore pivot tables and charts
